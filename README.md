# 📊 Excel Sales Dashboard

An interactive **Sales Performance Dashboard** built using **Microsoft Excel** to analyze sales data, monitor employee performance, and track business KPIs through dynamic charts, tables, and city-wise filtering.

This project demonstrates how Excel can be used to create an interactive Business Intelligence dashboard for effective sales analysis and decision-making.

---

# 📌 Project Overview

The dashboard provides a comprehensive overview of sales performance across multiple cities. It allows users to analyze sales executives' performance, compare total sales, evaluate target achievement, and identify employees who are away from their sales targets using interactive visualizations.

---

# 🎯 Project Objectives

✅ Analyze sales performance across different cities.

✅ Compare sales executives based on total sales.

✅ Measure Target Hit Percentage.

✅ Identify employees below their sales targets.

✅ Monitor business KPIs through interactive dashboards.

✅ Present data in a visually appealing and easy-to-understand format.

# 📂 Dataset Information

The dataset includes the following columns:

| 📌 Column | 📝 Description |
|-----------|----------------|
| 🆔 Emp Code | Employee ID |
| 👤 Sales Executive | Employee Name |
| 🌍 Region | Sales Region |
| 📅 Day1 - Day5 | Daily Sales |
| 💰 Total Sales | Total Sales Generated |
| 🎯 Target | Assigned Sales Target |
| 📈 Target Hit % | Percentage of Target Achieved |
| 📉 Away From Target % | Remaining Percentage to Reach Target |
